# 🛡️ RM_Sentinel: Real-Time Host-Based IDS & SOC Dashboard

**RM_Sentinel** هو نظام أمني متكامل لحماية الأجهزة المضيفة (HIDS) ومراقبة سلامة الملفات (FIM)، مربوط بغرفة عمليات أمنية ذكية (SOC Dashboard) لتعقب التهديدات جغرافياً وبث التنبيهات وإرسال الإشعارات بشكل فوري (Live Event & Threat Mapping).

---

## 🎯 بنية وهندسة النظام (System Architecture)

يعتمد النظام على معمارية العميل والسيرفر (Agent-Server Architecture) لتوفير أقصى درجات المراقبة دون التأثير على أداء الجهاز المضيف:
1. **العميل (Agent):** برنامج خفيف يعمل في الخلفية يراقب استهلاك الموارد (CPU/RAM) ويتعقب التهديدات ومحاولات الدخول العشوائي (Brute Force)، بالإضافة إلى نظام مراقبة سلامة الملفات (File Integrity Monitoring).
2. **الخادم المركزي (Server):** مبني باستخدام FastAPI لاستقبال اللوجات عبر نفق آمن (Ngrok)، معالجة البيانات، حساب معدلات الخطورة (Risk Scoring)، وحفظها في قاعدة بيانات PostgreSQL.
3. **غرفة العمليات الأمنية (SOC Dashboard):** واجهة تفاعلية مبنية بـ Streamlit تعرض عدادات لايف، مخططات مئوية للمخاطر، وخريطة جغرافية تفاعلية تحدد موقع المهاجم الفعلي بدقة وتضع دوائر الخطر الحمراء.

---

## ✨ المميزات الرئيسية (Core Features)

* **🔍 مراقبة سلامة الملفات (File Integrity Monitoring):** تتبع لحظي لإنشاء، تعديل، أو حذف الملفات داخل المجلدات الحساسة باستخدام مكتبة `Watchdog`.
* **🗺️ الخرائط السيبرانية التفاعلية (Geographical Threat Mapping):** سحب الأيبي الحقيقي للعميل والتحقق من موقعه الجغرافي (الإحداثيات والمدينة) ورسمها مباشرة على الخرائط التفاعلية (Dark Matter Theme).
* **📊 نظام تقييم المخاطر الذكي (Dynamic Risk Scoring):** معادلة برمجية مخصصة لحساب نسبة الخطر (تصل إلى 100%) بناءً على نوع الهجوم المكتشف واستهلاك الموارد.
* **🔔 الإشعار والاستجابة الفورية (Instant Incident Response):** ربط مباشر مع Telegram API لإرسال تقارير مفصلة فور حدوث أي خرق أمني تشمل (اسم الجهاز، نوع التهديد، نسبة الخطر، الموقع والوقت).

---

## 🛠️ الحزمة التقنية (Tech Stack)

* **الخلفية (Backend):** FastAPI (Python 3.12), SQLAlchemy
* **قاعدة البيانات (Database):** PostgreSQL (psycopg2)
* **واجهة العرض (Dashboard):** Streamlit, Folium (Maps), Streamlit-Folium, Plotly
* **مراقبة الأنظمة (Agent & FIM):** Watchdog, Psutil, Requests
* **الربط والنفق (Tunneling):** Ngrok (لتمرير البيانات من وإلى الخادم عبر الإنترنت)
